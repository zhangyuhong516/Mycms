K.colorpicker(options)¶

创建取色器。

  • 参数:
    • object options: 配置信息

  • 返回: KColorPicker

  • 继承: KWidget ( K.widget(options) )

示例:

var colorpicker = K.colorpicker({
        x : 100,
        y : 200,
        z : 1000,
        selectedColor : 'default',
        click : function(color) {
                alert(color);
        }
});
